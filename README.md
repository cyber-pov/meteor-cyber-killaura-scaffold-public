# Meteor Cyber KillAura + Scaffold

Contains:

- `KillAuraPlus`
- `ScaffoldPlus`

Also includes:

- `RotationManager`
- `RotationTarget`
- `MovementCorrection`
- `RotationTiming`
- client-side mixins used for packet rotation and movement correction

## Compatibility

- Minecraft: `1.21.11`
- Meteor Client: `1.21.11`
- Fabric Loader: `0.18.1+`
- Built with Fabric Loader: `0.18.4`

## Build

```bash
./gradlew build
```
